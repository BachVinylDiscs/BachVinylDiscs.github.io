Python

Unit 1

Description

Unit 1 was the first unit, and it introduced me to Python and the language. It involved strings and numbers, and classes and objects in a runner file. It used pygame to make a visual representation of objects.

Concepts Learned

  • How to work with numbers and strings
  • How to import classes and get objects from them
  • How to use the "pygame" module

Unit 2

Unit 2 and it mainly involved interaction between different objects. We imported classes to a runner file, wrote code to dictate how they interact. Collisions were common. When two objects' 'rectangles' intersected, then there was written code to make them behave in a certain way.

Concepts Learned

  • How to import and place many objects
  • Writing inter-object methods correctly
  • Writing text and doing other things to make a game user-friendly

Unit 3

Description

Unit 3 used "for" loops to loop through a method multiple times. We created randomly placed rocks by using for loops and setting each rock's position as random. We also made a game where the player moves around the screen, "collecting" a coin, which teleported to random locations on the screen every time it was "collected".

Concepts Learned

  • How to use "for" loops
  • How to assign random values to variables
  • How to keep a score

Unit 4

Description

Unit 4 used lists to create multiple objects of the same class, and manipulate as a group, not as individual objects. We continued using games as the medium of testing, such as creating multiple enemies, multiple costumes, etc.

Concepts Learned

  • How to use lists
  • How to more complex games
  • How to use lists to make the game mroe enjoyable
