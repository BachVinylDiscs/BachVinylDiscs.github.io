Final Project

"Coin Collector 2"

Description

My final project for the class was a game written in Python. I didn't create any new files; instead, I modified existing ones. There are 5 class files (Coin, Enemy, Enemies, Person, and Ground.) Ground was the same file used in earlier units. Enemy made the aliens change directions randomly every 5 seconds. If it went off the screen, it would teleport to a random location in the screen, with a random direction. They all move at the same speed. The Enemies file moved all the aliens at once, and set the speed for the aliens. The Person file got textures for the Person when it moved in different directions, and got methods to move it. The Coin file got the coins going, and there are 4 different textures for them (US Quarter, US Dime, US Nickel, US Penny). The aliens move around the screen, and when the Person collides with it, the Person loses. When the player collides with the Coin, the Person's score goes up by 1 (displayed on screen), and the aliens speed up a bit. When the score reaches 15, the player wins.

Concepts Learned

  • How to make a good game
  • How to create and use lists
  • How to debug programs (mainly incorrect syntax, etc.)
  • Use of random numbers requires me to import random at the top
  • setPos methods to change class __init__ values (self.x, self.y, self.cos and more)
