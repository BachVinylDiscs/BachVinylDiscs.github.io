Name Assignment

Description

Use Scratch to make a program that shows my name, and give each letter 3 unique behaviors. Program needed to run when the green flag was clicked, and extra credit was given when the screen reset each time the green flag was clicked.

Concepts Learned

  • Used event blocks (green flag clicked to start game)
  • Conditional statements (if statements; I used it to trigger scripts when something happened)
  • Used loops (repeating; I used it to repeat a small movement to make it move across the screen or rotate completely)
  • Moving Sprites (sprites for each letter that moved in unique ways)
